## Description

Mobile : Drones Management Project

## Installation

```bash
$ npm install
# or
$ yarn install
```


## Running the app

```bash
# development
$ npm run start

#android
$ npm run android

#ios
$ npm run ios

#web
$ npm run web

#WITH YARN

# development
$ yarn start

#android
$ yarn android

#ios
$ yarn ios

#web
$ yarn web
```
